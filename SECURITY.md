# Security Policy

## Supported Versions

Skein is pre-1.0; only the latest release receives security fixes.

## Reporting a Vulnerability

Please report vulnerabilities privately via
[GitHub Security Advisories](https://github.com/kormie/Skein/security/advisories/new)
rather than opening a public issue. You should receive a response within a
few days. Please include a minimal reproduction where possible.

Skein programs declare their effects via capabilities; reports about
capability-enforcement bypasses (code performing I/O its module never
declared) are especially valuable.
